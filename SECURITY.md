# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ITAM, please report it by:

1. **Email:** admin@designedbymerry.com
2. **GitHub:** Open a private security advisory

Please do NOT open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Security Updates

We take security seriously. Updates are released as needed.
